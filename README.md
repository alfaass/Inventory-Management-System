# Inventory Management System

## Mini Project

A Flask based Inventory Management System for small business.

## Features

- Add Product
- Edit Product
- Delete Product
- Search Product
- Record Sales
- Automatic Stock Update
- Low Stock Alert
- Sales Report

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JSON

## How to Run

Install Flask:

pip install -r requirements.txt

Run:

python app.py
